# Cole Downing — Portfolio

Personal portfolio site built with React and Tailwind CSS.

Website url: http://localhost:5173/#projects 

## Tech stack

- [React](https://react.dev) + [Vite](https://vitejs.dev)
- [Tailwind CSS v4](https://tailwindcss.com)

## Local setup

```
npm install
npm run dev
```
